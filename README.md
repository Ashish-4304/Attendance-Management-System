# Attendance Management System (Java Project)

## Overview
This is a simple Java OOP project demonstrating:
- Classes & Objects
- Methods
- Aggregation
- Collections
- Modular structure

## How to Run the Project
1. Open Command Prompt
2. Navigate to src folder  
   `cd path_to_your_project/src`
3. Compile the project  
   `javac *.java`
4. Run the project  
   `java Main`
